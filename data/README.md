# Data

Working datasets for analysis: CSV, JSON, or similar.

A file belongs here when it is under 10 MB and contains no personal,
financial, customer-level, or otherwise sensitive information. Everything
else lives in external storage with an entry in `client.yaml` →
`external_storage` — including anything that may need guaranteed deletion
later, because Git history is permanent.

Add a short note per dataset (here or alongside the file): what it is,
where it came from, when it was extracted.
