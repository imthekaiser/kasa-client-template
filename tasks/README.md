# Tasks

One YAML file per scoped agent task, named `{TASK-ID}.yaml` (for example
`ACME-WEB-001.yaml`). Copy the format from `examples/task.example.yaml` in
the `kasa-platform` repo. A task defines the objective, scope, inputs,
allowed repositories, deliverables, acceptance criteria, and permissions.

Agents work only within the task they are assigned.
